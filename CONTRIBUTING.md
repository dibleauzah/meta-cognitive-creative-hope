# CONTRIBUTING.md

# Contributing to Metacognitive Creative Hope (MCC-Hope)

Thank you for your interest in contributing.

MCC-Hope is an evolving open-source project focused on:

- metacognition
- self-communication
- operational resilience
- reflective systems
- human-centered cognition
- productivity architecture
- QA + Kaizen thinking
- open-source experimentation

The goal is not merely to build software, but to explore how humans think, adapt, self-regulate, and coordinate under pressure.

---

# Ways to Contribute

Contributions may include:

- code
- documentation
- UI/UX ideas
- accessibility improvements
- QA/testing
- diagrams
- research synthesis
- OpenStreetMap integrations
- mental-health resource mapping
- journaling workflows
- issue triage
- educational content

---

# Contribution Philosophy

Please optimize for:

- clarity
- kindness
- systems thinking
- maintainability
- accessibility
- usefulness under stress

This project values thoughtful iteration over perfection.

---

# Before Submitting

Please:

1. Search existing issues first
2. Keep pull requests focused
3. Explain reasoning clearly
4. Document meaningful changes
5. Avoid unnecessary complexity

---

# Important Disclaimer

MCC-Hope is not a replacement for licensed medical, psychiatric, or emergency services.

Contributors should avoid presenting the project as clinical treatment software unless appropriate licensed frameworks and compliance systems are formally established.
